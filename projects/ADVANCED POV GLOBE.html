Project description

Persistence of vision is an optical illusion in which many discrete images are blend into one single image on the human mind. This particular board has a set of 10 inline LEDs that you can program to display the POV effect. We here implemented a POV display based on Raspberry Pi 3 B+. Objective:

  • The Purpose of this project is design and create a persistence of vision display.
  • The display will allow the users the to upload an image to be displayed through wireless communication.
  • A persistence of vision refers to the phenomenon to the Lunan eye in which an after image exists for a brief time.
  • A POV display exploits this phenomenon by spinning in one dimension row of LEDâ€™s through a two dimensional space at such a high frequency that a two dimensional display is visible.
