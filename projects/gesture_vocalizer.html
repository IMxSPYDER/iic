Project description

Sign language is the language used as a way of communication for deaf and dumb people. It consists numerous signs or gestures instead of sound to convey the exact meaning. So, for this type of communication a concept of ‘Gesture Vocalizer’ is used.
This work consists of a glove equipped with sensors which senses different sign language gestures. Then the data collected by these sensors is fed to the Arduino. Then this data is further transferred to an android phone via Bluetooth Module. A common android phone is used for sign to text and speech translation.

Future Scope

In future dedicated application for gesture vocalizer can be developed in which the user can set his own gestures and shortcuts. User can customize different output voices and also can set his own voice for the gestures. Personal voice assistants such as Alexa, Google Assistant can be included in the glove which can also support some basic home automation shortcuts such as switching ON and OFF lights, increasing/decreasing the thermostat etc.
